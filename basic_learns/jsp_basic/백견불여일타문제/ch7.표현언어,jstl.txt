-------- ch7.표현언어,jstl --------
1. 	표현언어는 ${}를 통해서 어떠한 값을 출력하기 위해 사용되는 문법
	표현언어는 JSP에서 자바코드와 HTML코드가 동시에 쓰이면 가독성이 떨어지는 것을 보완
	
==>
표현 언어는 값(데이터)을 웹 페이지에 표시(표현)하는데 사용되는 태그
--------------------------------------------------------
2.	JSTL = java standard tag library 
	core -> 변수 설정 및 조건 분기 등등
	fmt	-> 데이터 포맷팅
	sql	-> 디비 관련

==>
JSTL 태그 라이브러리는 JSP는 스크립트릿과 자바 코드 등 여러 코드들이 섞여서 복잡한
구조로 되어있는데, 간결하고 이해하기 쉽게 코딩을 하기 위해 추가한 태그
--------------------------------------------------------
3.	html과 함께 쓰일 때 가독성이 높아짐
