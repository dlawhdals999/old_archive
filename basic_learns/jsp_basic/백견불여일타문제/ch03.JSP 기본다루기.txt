처음 요청에는 jsp페이지를 바꾼 자바의 클래스의 인스턴스 생성 
이후 요청에는 생성된 인스턴스로 요청 처리

==>
브라우저에서 JSP 페이지 ("....jsp")를 웹 서버에게 요청
>웹 서버는 JSP에 대한 요청을 서블릿 컨테이너에게 넘김 
>웹 컨테이너는 해당 JSP를 찾아서 서블릿 파일(..._jsp.java)를 생성
>서블릿 파일은 컴파일되어 서블릿 클래스 파일(..._jsp.class)생성
>컴파일된 서블릿 클래스 파일을 웹 서버로 보냄
>사용자는 서블릿의 출력 결과인 HTML 형태로 응답 결과를 보게 됨
